# C basic knowledge

```c
spin()
```

等一秒钟返回

```c
assert( 表达式 );
```

若表达式值为0，则终止程序

```c
malloc(num);
int *p=malloc(num);
```

动态分配内存函数，分配长度为num字节的内存块，需要使用free释放。

p是一个指针变量，表示内存中的一个地址。*p表示这个地址内存中存的值。
